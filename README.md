# reviewcheck

楽天店舗向け：商品レビュー＆画像チェックツール

## 概要

楽天市場店舗の自社商品について、以下を一覧で素早く確認できるWebアプリです：

- **商品画像の1枚目・2枚目・4枚目**(500×500の高解像度)
- **レビュー件数・平均評価**(★表示)
- **画像付きレビューへのリンク**(楽天本サイトを別タブで開く)
- **チェック済みフラグ**(進捗管理)

複数ショップ対応、GitHub PATでのデータ同期(全PCで共通)対応。

## 使い方

### 初期設定

1. **⚙ 設定** ボタンを開く
2. 以下を入力:
   - **アプリケーションID** (楽天Developers アプリ一覧)
   - **アクセスキー** (アプリ一覧の目アイコンで表示)
   - **店舗コード(shopCode)** (例: `yukaiya` ※楽天市場URL内の英数字)
   - **GitHub PAT** (任意・データ共有時のみ・repo権限)
3. **接続テスト** で楽天APIが疎通することを確認
4. **保存**

### データ取得

1. 上部の **🔄 商品データ取得** ボタンを押す
2. 楽天Ichiba Item Search APIから店舗の全商品を取得(1.2秒間隔)
3. 100商品で約2分、300商品で約6分

### レビューチェック

1. 商品カードに表示される星(平均評価) + 件数を確認
2. **📷 画像付きレビューを見る** ボタンで楽天本サイトのレビューページを別タブで開く
3. 画像付きレビューもそこで確認できる(楽天APIではレビュー本文・画像取得不可のため)
4. チェック済みになったら **○ 未チェック** ボタンを押して **✓ チェック済** にする

### 画像拡大

商品カードの画像をクリック → ライトボックスで拡大表示

### 検索・絞り込み・並び替え

- 商品名・商品番号で検索
- フィルタ: 全て / 未チェック / チェック済み / レビューあり
- 並び順: レビュー数・平均評価・商品名

## 技術仕様

### フロントエンド
- 単一HTMLファイル(HTML+CSS+JavaScript)
- フレームワークなし(Vanilla JS)
- フォント: Noto Sans JP + DM Mono

### バックエンド
- 楽天Ichiba Item Search API (`openapi.rakuten.co.jp/ichibams/api/IchibaItem/Search/20220601`)
- GitHub Contents API (PAT認証で `data/{shopId}/review_data.json` に保存)

### レート制限対応
- 楽天API: 1.2秒間隔(SAFE_INTERVAL=1200ms)
- 429エラー3回連続で中断(既存データ保護)

### 画像高解像度化
楽天APIの `mediumImageUrls[]` は通常128×128。URL末尾の `?_ex=128x128` を `?_ex=500x500` に置換して500pxで表示。

### 画像付きレビュー取得について

**⚠ 楽天ウェブサービスAPIには商品レビューを取得するAPIが存在しません。**
レビュー件数(`reviewCount`)と平均評価(`reviewAverage`)はAPIで取得可能ですが、レビュー本文・画像はAPI公開外のため、画像付きレビューを確認する場合は楽天本サイトのレビューページに遷移する設計です。
